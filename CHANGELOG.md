# Changelog

## v0.3.0
- **Откат соцрейтинга** — снять начисления прямо с доски: всё разом, выборочно (галочки) или конкретного ника (инлайн `↩`). Стейтлес: читается по меткам на доске, не зависит от памяти расширения.
- Метки теперь хранят сумму — `[СР] ник:100` — это и нужно для точного отката.
- Режим **«как донат»** (`isDonation`): ставка в донат-стиле, к сумме применяется конвертация аука.
- Кнопка «Откатить» активна только когда на доске есть метки (показывает их число); пофикшена гонка, из-за которой она иногда не включалась после залива.

## v0.2.0
- Первый рабочий релиз.
- Залив рейтинга из Google-таблицы: авто «+ к лоту» для одиночного вкладчика; спорные / групповые / без лота — выбор лота в окне.
- Метки-инвесторы `[СОЦРЕЙТИНГ] ник` на лоте + защита от задвоения («уже залито»).
- Плюс/минус баллы, пропуск нулей, настройка столбцов / первой строки / префикса.
- Фоновая заливка одним батчем (окно расширения можно закрыть).
