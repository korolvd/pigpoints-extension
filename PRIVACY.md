# Политика конфиденциальности — PigPoints

_Обновлено: 29 июня 2026_

PigPoints — расширение Chrome для стримеров, которое превращает значки чата Twitch, баллы канала и данные из Google-таблицы стримера в ставки в его аукционе на [pointauc.com](https://pointauc.com/).

## Какие данные обрабатываются

Расширение работает с данными, которые ты сам подключаешь:

- **Токены подключений** — Personal Token pointauc и токен авторизации Twitch (OAuth), а также логин/ID твоего канала.
- **Настройки** — ссылка на таблицу, столбцы, URL и секрет Apps Script, пул значков и наград.
- **Данные зрителей в рамках механики** — twitch-логины и значения PigPoints (из твоей Google-таблицы и из активаций наград на твоём канале).

## Где данные хранятся

Все перечисленные данные хранятся **локально в твоём браузере** (`chrome.storage.local`). Расширение **не имеет собственного сервера** и **не отправляет твои данные разработчику**.

## Куда данные передаются

Только тем сервисам, которые ты подключаешь, и только для работы функций:

- **pointauc.com** — отправка ставок в твой аукцион через официальный API (с твоим Personal Token).
- **Twitch** (`api.twitch.tv`, `id.twitch.tv`) — авторизация, создание/управление наградами за баллы канала, чтение статусов зрителей и значков чата.
- **Google** (`docs.google.com`, `script.google.com`) — чтение твоей таблицы (CSV-экспорт) и запись начислений PigPoints через **твой собственный** Apps Script Web App.

Токены передаются только их собственным сервисам (Twitch-токен — на Twitch, pointauc-токен — на pointauc).

## Чего расширение НЕ делает

- Не передаёт и **не продаёт** данные третьим лицам.
- Не использует данные для целей, не связанных с работой расширения.
- Не содержит рекламы, аналитики и трекеров.

## Контроль и удаление

Отключить подключения и очистить журналы можно прямо в расширении. Удаление расширения из Chrome удаляет все локально сохранённые данные.
