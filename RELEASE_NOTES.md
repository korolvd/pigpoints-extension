# PigRating v0.3.0

Соцрейтинг свиноколеса для [pointauc.com](https://pointauc.com/): перед прокрутом заливает баллы из Google-таблицы в лоты через официальный API.

## Что умеет
- **Залив рейтинга:** авто «+ к лоту» для одиночного вкладчика; спорные / групповые / без лота — выбор лота прямо в окне (выпадающий список).
- **Метки с суммой:** на лоте появляется подпись `от [СР] ник:100` — видно, что начислено рейтингом; защита от задвоения («⚠ уже залито»).
- **Откат соцрейтинга:** всё разом, выборочно (галочки) или конкретного ника (инлайн `↩`). Стейтлес — читается прямо с меток на доске.
- **Гибкость:** плюс/минус баллы, режим «как донат», настройка первой строки / столбцов / префикса.
- **Фоновая заливка** одним батчем — окно расширения можно закрыть, дойдёт до конца.

## Установка
Скачайте `pigrating.zip` → распакуйте → `chrome://extensions` → Режим разработчика → «Загрузить распакованное» → выберите папку. Вкладку аукциона держите открытой (API шлёт команды в неё).

Подробнее — в [README](https://github.com/korolvd/pigrating-extension#readme).
